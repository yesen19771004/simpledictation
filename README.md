# 英语听写练习

一个纯前端的英语听写应用，打开浏览器即可使用。输入英文原文后，系统会自动分句，使用浏览器语音合成（TTS）朗读句子，用户听写并逐词核对，正确锁定、错误补填。

---

## 功能

- **自动分句**：粘贴英文原文后自动按句切分
- **TTS 朗读**：使用浏览器内置语音引擎朗读当前句子，支持调节语速（0.5x ~ 1.5x）
- **智能对齐**：提交后使用 LCS 算法自动对齐原文与输入，正确的单词锁定，缺失或错误的单词变成独立空格
- **原文与翻译**：随时查看原文、句子翻译（MyMemory 免费接口）及逐词翻译
- **进度保存**：听写进度自动保存到浏览器 localStorage，关闭页面后可继续练习
- **重练支持**：完成后可清空进度重新练习

---

## 快速开始

1. 用浏览器打开 `index.html`（建议使用 Chrome / Edge / Safari）
2. 点击 **新建练习**，输入标题和英文原文
3. 点击 **生成练习**，进入听写页面
4. 点击 **播放** 听句子，在输入框中写下听到的内容
5. 点击 **提交检查**，对的单词会被绿色锁定，错的变成空格
6. 在空格中填入正确单词，再次提交，直到全对进入下一句

## 使用说明

### 播放控制
- **▶ 播放**：朗读当前句子
- **⏹ 停止**：播放中可随时点击打断，打断后可重新播放
- **语速滑块**：调节朗读速度

### 听写流程
1. **首次提交**：在文本框中输入听到的整句内容并提交
2. **补全空格**：系统会自动比对原文，在下方生成一排词（锁定的显示原文，空格显示为输入框）
3. **逐空填写**：在空格里填入正确单词，再次提交
4. **全对过关**：所有词正确后自动进入下一句

### 查看帮助
- 点击 **显示原文** 展开面板，可查看：
  - **原文**：当前句子的完整原文
  - **句子翻译**：自动翻译为中文
  - **单词翻译**：点击任意单词查看释义

### 进度管理
- 练习过程中可随时点击 **上一句 / 下一句** 切换，草稿会自动保存
- 回到首页后，练习会显示在列表中，点击 **继续** 即可恢复进度
- 点击 **重练** 会清空该练习的进度，从头开始

## 浏览器兼容性

- ✅ Chrome / Edge（推荐）
- ✅ Safari
- ❌ Firefox（语音合成支持较弱，可能无法播放）

> **注意**：语音合成功能依赖浏览器的 `speechSynthesis` API，首次使用可能需要联网加载语音数据。

---

## 数据存储

所有数据保存在浏览器的 `localStorage` 中，**不会上传到服务器**。清除浏览器数据会导致练习记录丢失。
