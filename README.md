# 英语听写练习

一个纯前端的英语听写应用，打开浏览器即可使用。输入英文原文后，系统会自动分句，使用浏览器语音合成（TTS）朗读句子，用户听写并逐词核对，正确锁定、错误补填。

内置 **180 篇原创听写文本**，覆盖 CEFR A1-C2 六个级别，无需手动输入即可一键开始练习。

---

## 功能

- **练习库**：内置 180 篇按 CEFR 分级的原创听写文本（A1/A2/B1/B2/C1/C2 每级 30 篇），一键导入开始练习
- **自动分句**：粘贴英文原文后自动按句切分
- **TTS 朗读**：自动选择高质量英语语音朗读，支持调节语速（0.5x ~ 1.5x）
- **智能对齐**：提交后使用 LCS 算法自动对齐原文与输入，正确的单词锁定，缺失或错误的单词变成独立空格
- **原文与翻译**：随时查看原文、句子翻译（MyMemory 免费接口）及逐词翻译
- **进度保存**：听写进度自动保存到浏览器 localStorage，关闭页面后可继续练习
- **重练支持**：完成后可清空进度重新练习

## 快速开始

### 方式一：从练习库开始（推荐）

1. 用浏览器打开 `index.html`（建议使用 Chrome / Edge / Safari）
2. 点击 **练习库**，选择适合你的 CEFR 级别
3. 点击任意课程标题旁的 **开始练习**
4. 进入听写页面，点击 **▶ 播放** 听句子并写下听到的内容
5. 点击 **提交检查**，对的单词锁定，错的变成空格
6. 在空格中补全，再次提交，直到全对进入下一句

### 方式二：手动创建练习

1. 打开 `index.html`
2. 点击 **新建练习**，输入标题和英文原文
3. 点击 **生成练习**，后续步骤同上

## 使用说明

### 练习库

- **按级别筛选**：点击 A1 / A2 / B1 / B2 / C1 / C2 标签快速切换难度
- **进度追踪**：每篇显示完成状态（未开始 / 进度 X% / 已完成）
- **继续练习**：已开始的课程显示 **继续** 按钮，进度自动恢复
- **重置进度**：点击 **重置** 可清空该课程进度，从头再来

### 播放控制
- **▶ 播放**：朗读当前句子（自动选择系统最佳英语语音）
- **⏹ 停止**：播放中可随时点击打断，打断后可重新播放
- **语速滑块**：调节朗读速度

### 听写流程
1. **首次提交**：在文本框中输入听到的整句内容并提交
2. **补全空格**：系统会自动比对原文，在下方生成一排词（锁定的显示原文，空格显示为输入框）
3. **逐空填写**：在空格里填入正确单词，再次提交
4. **全对过关**：所有词正确后自动进入下一句

### 查看帮助
- 点击 **显示原文** 展开面板，可查看：
  - **原文**：当前句子的完整原文
  - **句子翻译**：自动翻译为中文
  - **单词翻译**：点击任意单词查看释义

### 进度管理
- 练习过程中可随时点击 **上一句 / 下一句** 切换，草稿会自动保存
- 回到首页后，练习会显示在列表中，点击 **继续** 即可恢复进度
- 点击 **重练** 会清空该练习的进度，从头开始

## 练习库内容

| 级别 | 难度 | 每篇长度 | 主题范围 |
|------|------|----------|----------|
| **A1** | 初学者 | 60-80 词 | 家庭、天气、学校、日常对话 |
| **A2** | 初级 | 100-120 词 | 旅行、周末、生日、购物、健康 |
| **B1** | 中级 | 150-180 词 | 社会话题、文化差异、科技、环境 |
| **B2** | 中高级 | 200-250 词 | 深度论述、经济、政治、伦理争议 |
| **C1** | 高级 | 250-300 词 | 学术分析、哲学、语言学、国际关系 |
| **C2** | 精通 | 300-350 词 | 复杂理论、批判性思维、前沿科技伦理 |

所有文本均为原创，覆盖日常对话到学术思辨，适合长期系统练习。

## 浏览器兼容性

- ✅ Chrome / Edge（推荐）
- ✅ Safari
- ❌ Firefox（语音合成支持较弱，可能无法播放）

> **注意**：语音合成功能依赖浏览器的 `speechSynthesis` API，首次使用可能需要联网加载语音数据。应用会自动选择系统中质量最高的英语语音，避免默认哑音问题。

---

## 数据存储

所有数据保存在浏览器的 `localStorage` 中，**不会上传到服务器**。清除浏览器数据会导致练习记录丢失。
