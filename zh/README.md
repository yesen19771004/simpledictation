# Dictation — 中文听写练习

> 🌐 **在线体验：[https://tx.comdosoft.com](https://tx.comdosoft.com)**
> 
> 所有数据存储在浏览器本地，**无需注册**，打开即用。

一个纯前端的**中文听写应用**，打开浏览器即可使用。输入中文原文后，系统自动分句，使用浏览器语音合成朗读句子，用户听写并逐字核对，正确锁定、错误补填。

内置 **52 篇原创中文听写文本**，覆盖 A1-B2 四个难度级别。同时支持粘贴自己的文本创建自定义听写，以及导入外部资料库。

界面支持**中英文切换**，点击顶部语言按钮即可。

---

## 功能亮点

- **资料库**：52 篇按难度分级的原创中文文本（A1-B2），一键开始
- **多资料库**：支持导入外部资料库，多个资料库自由切换
- **数据迁移**：听写记录与错题数据支持导出/导入，换设备不丢失
- **专项训练**：自动追踪听写出错的薄弱词汇，生成 LLM 提示词，让 AI 量身打造针对性训练文章
- **逐字对齐**：贪心匹配算法自动比对原文与输入，正确字锁定，错误变为空格
- **显示语言切换**：界面可在中文/英文间切换，点击顶部语言按钮即可
- **快捷键支持**：全程可键盘操作，无需鼠标
- **进度保存**：听写进度自动保存到浏览器 localStorage

## 快速开始

### 方式一：从资料库开始（推荐）

1. 用浏览器打开 `index.html`（建议使用 Chrome / Edge / Safari）
2. 点击 **资料库**，选择适合你的难度级别
3. 点击任意课程旁的 **开始练习**
4. 点击 **▶ 播放** 听句子，在输入框写下听到的内容
5. **Ctrl + Enter** 提交检查，对的字锁定，错的变成空格
6. 在空格中补全，再次提交，直到全对自动进入下一句

### 方式二：手动创建听写

1. 打开 `index.html`
2. 点击 **新建听写**，输入标题和中文原文
3. 点击 **生成练习**，后续步骤同上

## 使用说明

### 资料库

- **按级别筛选**：A1 / A2 / B1 / B2 快速切换难度
- **进度追踪**：每篇显示完成状态（未开始 / 进度 X% / 已完成）
- **继续 / 重置**：已开始的课程可继续或清空进度重来
- **资料库切换**：支持导入外部资料库，在顶部切换栏选择不同资料库
- **删除资料库**：导入的资料库可删除，默认资料库不可删除

### 导入资料库

资料库文件为 JSON 格式，结构如下：

```json
{
  "name": "资料库名称",
  "lessons": [
    {
      "id": "unique-id-01",
      "level": "A2",
      "title": "文章标题",
      "text": "中文原文内容..."
    }
  ]
}
```

每个课程需包含 `id`、`level`、`title`、`text` 四个字段。

### 我的数据

在「我的听写」页面底部可管理个人数据：

- **导出我的数据**：将听写记录和错题数据导出为 JSON 文件备份
- **导入我的数据**：从 JSON 文件恢复数据，支持「覆盖」或「合并」两种模式
- **清空数据**：一键清空所有听写记录、错题记录和资料库映射（操作前请确认已导出备份）

### 播放

- **▶ 播放**：朗读当前句子
- **⏹ 停止**：播放中可随时打断
- **语速滑块**：0.5x ~ 1.5x 调节

### 听写流程

1. **首次提交**：输入整句并提交
2. **补全空格**：系统自动比对，正确字锁定，错误字变为输入框
3. **逐空填写**：填入正确字词，再次提交
4. **全对过关**：所有字正确后自动进入下一句

### 快捷键

| 快捷键 | 作用 |
|--------|------|
| **Enter** | 焦点不在输入框时，聚焦到第一个输入框 |
| **Space** | 焦点不在输入框时，播放/暂停 |
| **Shift + Space** | 焦点在输入框内时，播放/暂停 |
| **Ctrl + Enter** | 提交检查 |

### 查看辅助

- 点击 **显示原文** 展开面板：
  - **原文**：当前句子完整原文
  - **句子翻译**：自动翻译为英文（MyMemory 接口）

### 专项训练

系统会自动记录你在常规听写中出错的字词，并统计出错次数。

1. 进入 **专项训练** 页面
2. 查看错题记录和待训练词汇数量
3. 点击 **生成提示词**，选择适合的难度级别（A1-B2）
4. 复制提示词到任意 LLM（ChatGPT、Claude 等）
5. 将 LLM 生成的文章粘贴回来，点击 **开始专项训练**
6. 逐句听写，每句仅一次机会
7. 核对后，已记录的错词写对则出错次数减 1（≤0 删除），写错则加 1

## 数据存储

所有数据保存在浏览器的 `localStorage` 中，**不会上传到服务器**。清除浏览器数据会导致练习记录丢失。建议定期使用「导出我的数据」功能进行备份。

## 技术说明

### 分句

中文分句按 `。！？；` 分割，同时支持英文标点 `.!?`，不做缩写保护。

### 字符比对

每个汉字（含标点）作为一个独立 token。采用贪心正向匹配算法逐字比对：

- 原文的每个字从用户输入的当前位置往后查找
- 找到则标记匹配，未找到则标记为空格
- 用户漏写的字会显示为输入框，需要补填

### 数字归一化

核对时，中文数字与阿拉伯数字视为等价：

| 原文 | 用户写 | 判定 |
|------|--------|------|
| `五` | `5` | ✅ 正确 |
| `2020` | `二〇二〇` | ✅ 正确 |

### 显示语言切换

界面文字支持中文/英文切换，默认为中文。点击顶部语言按钮即可切换。语言偏好保存在 localStorage 中。
